# ai moms™ × Claude Code — One-Page Cheat Sheet

> Keep this next to you. You don't write code. You describe what you want, Claude builds it,
> you approve it. Your job is to be a clear, decisive boss.

## The 5 things to remember
1. **Just talk to it.** Plain English. "Add a new article page about meal planning."
2. **Read the plan before saying yes.** Approving is a real decision, not a formality.
3. **One change at a time**, then look at the site to check.
4. **Save often.** Ask it to "save this version" — that's your undo button.
5. **Confused? Ask it to explain simply.** It's a patient teacher, not just a builder.

## Words you'll hear (plain English)
| Word | What it really means |
|---|---|
| **Front end** | What visitors see — the pages. |
| **Back end** | The hidden part that does work and talks to the database/AI. |
| **Database (Supabase)** | A super-spreadsheet where your articles and data live. |
| **ORM** | A translator so code can talk to the database in plain commands. |
| **Authentication** | Login — how the site knows who someone is. |
| **Testing** | Automatic safety checks before anything goes live. |
| **GitHub** | Cloud backup + full history of your site. Your safety net. |
| **Deployment** | Publishing changes to the live internet (Netlify does this). |
| **Commit** | "Save this version." | 
| **Push** | "Publish it" → the live site updates. |
| **Revert** | "Undo — go back to the last good version." |
| **CLAUDE.md** | Your house manual that Claude reads every time. |
| **Plan Mode** | Claude proposes before it builds. Your seatbelt. |

## Good ways to ask
- ✅ "Fix the typo in the second paragraph of the homepage."
- ✅ "Make a plan first: I want a new mini-tool that suggests birthday party themes."
- ✅ "That's not quite right — make the button calmer and smaller. Try again."
- ✅ "Explain what just happened, like I'm new to this."
- ✅ "Save this version" / "Undo that, go back to before."

## When to STOP and ask the engineer
- It wants to change the live site / domain and you're not sure.
- It mentions deleting data or a database table.
- Something on the live aimoms.ai looks broken.
- It asks for a password or API key — never paste secrets into chat.

## Your daily loop
**Idea → ask Claude → read its plan → approve → look at the preview → "save this version."**
That's the whole job. Repeat.
